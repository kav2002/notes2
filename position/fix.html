Positions

The position property specifies the type of positioning method used for an element.
Element are then postioned using top, bottom, left, right properties.

  • static
  • relative
  • fixed
  • sticky

Fixed Position

An element with position: Fixed; it is always stays in the same place even if the page is scrolled.A fixed element does not leave a gap in the page where it would normally have been located.
